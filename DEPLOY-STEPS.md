# Deploy Steps

1. Upload every file/folder from this package to the root of your GitHub repo.
2. Replace existing files when GitHub asks.
3. Commit changes.
4. Vercel will redeploy automatically, or press Redeploy manually.
5. Make sure Vercel Environment Variables exist:
   - VITE_SUPABASE_URL
   - VITE_SUPABASE_ANON_KEY
